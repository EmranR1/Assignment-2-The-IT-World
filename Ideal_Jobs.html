Ideal Jobs 

 

Comparison of ideal jobs:

Cyber security specialist, Cybersecurity analyst and Penetration tester are all cyber security focused. Cyber security specialists and Cybersecurity analysts have very similar responsibilities. Penetration testing is also in the Cybersecurity field. These roles all have very high pay, with a cybersecurity specialist being the highest with over +100k salary in the long term. Some of these “ideal jobs” are actually merely entry level jobs, however they do have a very impressive salary which almost competes with the +100k salary of cybersecurity specialists. This is really good as cybersecurity specialists require tons of experience in many of these fields to be able to work as a cybersecurity specialist.

 

Where a cyber security specialist may provide a variety of cyber security services such as monitoring and investigating systems and maintaining security. A cyber security analyst might work on reports of existing systems and system ideas, to create recommendations to a company on how to improve and  secure their system and business from potential exploits / attacks. Penetration testing is about the application of cyber security analytics, through testing of security systems.In this way, all of these jobs are related with slightly different specialties. 

 

Cyber security specialist is a late stage career job with many years of experience required. Cyber security analyst and penetration testing meanwhile require some experience in IT, but can lead to many other jobs.

Daniel’s Ideal Job:

Penetration testing is a growing field, with an expected 47,300 ICT employees by 2023. Security related jobs are not on the burning glass top titles job list 2018.  

 

General skills like problem solving, communication and ability to learn are essential in these jobs. As are organisation, teamwork, report writing, planning, research, analytical skills and decision making. However more specifically computer skills are a requirement. Programming languages are very useful to know but can depend on the specific job, for example html, css and sql are all useful languages if a website were to be part of a more digital. On the other hand skills like lock picking and social engineering are useful for more physical penetration testing jobs. Capgemini for example requires basic understanding of AWS and Azure cloud (IaaS, PaaS, SaaS) knowledge of deploying Security tools in cloud environment and Good written & verbal communication skills among other things.   Generally due to the variety of possible jobs, penetration testing can require a large skillset, but can still be accomplished with a smaller one. In general, programming languages of varying kinds, windows and linux skills and systems engineering skills are required and useful.  

 

The skills used can be learned and used from a variety of places in a variety of ways. Windows PowerShell for example can be used to assist a system administrator in remotely managing company computers, however it can equally be used to steal data from an unsuspecting computer. Overall it is safe to say that all of the skills used as a penetration tester could equally be used in a variety of other IT related jobs, as such the demand for their skills is high for general IT positions and especially high for information security positions. The general communication, self learning and problem solving skills are sought out by many employers. They can be simplified to initiative and personable skills, which are required for many jobs, meaning from a non-IT related skill set these skills are fit for almost any job.  

 

The top three IT related skills that aren’t required for penetration testers are business management, project management and graphic design. The top three generic skills that aren’t required for penetration testers are troubleshooting, creativity and leadership.  

After looking at the burning glass data, I would still want to be a penetration tester because it is a job that covers a wide variety of topics, is paid on commission most of the time and relates to cyber security which is my interest.

Pauls Ideal Job:

Cyber Security analyst

As a security analyst, the job requires you to protect IT infrastructures (including networks, hardware and software) from various criminal activities, vulnerabilities, exploits and attacks. A cyber security analyst will monitor networks and systems, detect security threats ("incidents"), analyze and evaluate alerts, and report threats, intrusion attempts, and false alerts through resolution or escalation based on severity.

The analysts job is to basically overlook an organisations system and analyse the system and vind issues and vulnerabilities that need to be addressed and fixed up. The analyst will provide a report to the organisation and provide them with solutions to patch the holes in the system to ensure maximum security and that data redundancy and leakage is minimised. Making sure that the system is secure against malicious attacks from third parties that aim to cause havoc on the system with a variety of methods and attacks.

Some very important technical  skills that a cyber security analyst has are Reverse engineering. To fix bugs and perform malware analysis, security analysts must disassemble the code and evaluate the software. Another important skill is Application design and understanding how the UI can be exploited, Security analysts must create security architectures and interfaces within the application and test them throughout the software development lifecycle to ensure that the system is secure and to minimise data breaches, loss and redundancy. Another important technical skill an analyst must have is Firewall management. Security analysts must install and maintain firewalls to filter incoming traffic and keep track of access to the organization's computer network ensuring there are no untrustworthy mac addresses connected to the network for example. This requires a thorough understanding of the hardware and software firewall mechanisms and infrastructure.

Demand for Cyber security Analysts statistics. Referencing A1. cybersecurity jobs are among the most rapidly growing career areas. It is predicted that cybersecurity jobs will grow 31% through 2029, which is over seven times faster than the average economic job growth of 4%. With a constantly evolving industry with more and more security vulnerabilities arising every day there will be no shortage of jobs in the industry.
